## API
  
| 属性 | 说明 | 类型 | 默认值 |
| --- | ---  | --- | ---   |
| dot | 不展示数字，只有一个小红点 | boolean | false |
| count | 展示的数字，大于 overflowCount 时显示为 ${overflowCount}+，为 0 时隐藏 | number | - |
| overflowCount | 展示封顶的数字值 | number | - |
| showZero | 当数值为 0 时，是否展示 Badge | boolean | false |
