## 何时使用

| 属性         | 说明                 | 类型                                                                       | 默认值    | 版本 |

| src         | 图片类头像的资源地址    | string                                                                     |           |      |
| icon        | 设置头像的自定义图标   | React.ReactNode                                                             |          |      |
| size        | 设置头像的大小        | number                                                                     |            |      |
