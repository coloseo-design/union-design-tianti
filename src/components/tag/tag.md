## API
  
| 属性 | 说明 | 类型 | 默认值 |
| --- | ---  | --- | ---   |
| color | 标签色 | string | - |
| closable | 标签是否可以关闭 | boolean | false |
| visible | 是否显示标签 | boolean | true |
| onClose | 关闭时的回调 | (e: React.MouseEvent<HTMLElement>) => void | - |
| icon | 标签前插入icon | ReactNode | - |
