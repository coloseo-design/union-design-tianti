进度条

## 何时使用

在操作需要较长时间才能完成时，为用户显示该操作的当前进度和状态。

当一个操作会打断当前界面，或者需要在后台运行，且耗时可能超过 2 秒时；

当需要显示一个操作完成的百分比时。

## API

## Progress

| 属性 | 说明 | 类型 | 默认值 | 版本 |
| --- | --- | --- | --- | --- |
| percent | 百分比 | number | 0 |  |
| status | 状态，可选：success exception normal | string | normal |  |
| size | 大小 | string | - |  |
